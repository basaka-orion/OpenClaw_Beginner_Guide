> **难度**：⭐ 入门 | **阅读时间**：5分钟
> **一句话总结**：聪明的活用聪明的脑子，简单的活用便宜的脑子。省下 50% 的钱。

## 又是一个故事

你开了一家餐厅，有三个厨师：

| 厨师 | 特长 | 时薪 |
|------|------|------|
| 👨‍🍳 米其林大厨 | 法式料理，每道菜都是艺术 | 1000 元/小时 |
| 🧑‍🍳 家常菜师傅 | 快！好吃！量大！ | 100 元/小时 |
| 👩‍🍳 凉菜阿姨 | 拌个沙拉切个水果 | 30 元/小时 |

如果你**只请米其林大厨**做所有的菜——

- 切水果？大厨切。1000 元/小时。💀

**你每月 80% 的工资，花在了大厨切水果上。**

## AI 的世界也是一样

AI 模型就像厨师，有贵有便宜：

```
🧠 最聪明（最贵）                    💡 够用就行（最便宜）
   │                                      │
   ▼                                      ▼
Claude Opus ← 深度推理、复杂分析     Gemini Flash ← 日常写作、简单问答
DeepSeek V3.2 ← 中文推理、编程        MiniMax M2.7 ← 中文聊天、创意
```

### Token 是什么？

**Token ≈ 钱。** AI 每读一个字、写一个字，都在花 Token。

- 聪明的模型：每个 Token 贵
- 便宜的模型：每个 Token 便宜

### 🎯 省钱的核心逻辑

> **问自己一个问题：这个任务需要"思考"还是"执行"？**

| 需要"深度思考"的任务 | 用什么 | 费用 |
|:---------------------|:-------|:-----|
| 分析一份商业报告 | 🧠 最强模型 | ★★★★★ |
| 设计系统架构 | 🧠 最强模型 | ★★★★★ |

| 只需要"执行"的任务 | 用什么 | 费用 |
|:-------------------|:-------|:-----|
| 写一篇公众号文章 | ⚡ 快速模型 | ★ |
| 画一张海报 | 🎨 画画模型 | ★★ |
| 翻译一段话 | ⚡ 快速模型 | ★ |

## 🍕 超级省钱技巧：零轮询

这是最酷的技巧，用一个外卖的故事来解释——

### 以前的方式（轮询）：

```
你叫了外卖（让 AI 写代码）

你每 5 秒打一次电话：
  "好了吗？" — "没好"
  "好了吗？" — "没好"
  "好了吗？" — "没好"
  "好了吗？" — "没好"
  "好了吗？" — "好了！"

打了 100 通电话 ← 每通电话都要花 Token💀
```

### 现在的方式（零轮询 Hooks）：

```
你叫了外卖（让 AI 写代码）

你挂了电话去干别的事

外卖好了 → 自动门铃响 → 你去取

只花了 1 通电话的钱 ✅
```

**原理**：Claude Code 写完代码后，自动：
1. 📄 把结果写到一个文件里（相当于放进快递柜）
2. 🔔 给 OpenClaw 发通知（相当于按门铃）

你的 AI 管家被"叫醒"，读文件，然后告诉你"代码写好啦！"

## 📝 总结

| 省钱策略 | 效果 |
|---------|------|
| 不同任务用不同模型 | 省 50-70% |
| 零轮询 Hooks | 编程任务省 90%+ Token |
| Prompt 越短越好 | 每轮对话少花 Token |
