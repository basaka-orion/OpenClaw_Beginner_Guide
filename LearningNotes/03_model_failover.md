# 🛡️ 第3课：模型挂了怎么办？容灾保命指南

> **难度**：⭐⭐ 进阶 | **阅读时间**：4分钟
> **一句话总结**：一个模型挂了？没关系，自动换下一个，用户完全感觉不到。

---

## 什么是容灾？

想象你在高速公路上开车：

```
正常情况：
  ════════ 主路 ════════ → 到达 ✅

主路堵了：
  ════╳═══ 主路 ═══╳═══ → 堵住了 ❌
       ↘
        ════ 辅路 ════ → 到达 ✅  （自动绕道）

辅路也堵了：
  ════╳═══ 主路 ═══╳═══
       ↘
        ════╳ 辅路 ╳═══
             ↘
              ═══ 乡间小路 ═══ → 到达 ✅  （再绕一次）
```

**容灾 = 主路不通，自动走辅路。辅路也不通，再走小路。总之一定要到达目的地。**

---

## AI 模型也会挂

AI 模型不是永远在线的，它可能因为：

| 原因 | 多久会遇到 |
|------|-----------|
| 🔧 服务器维护 | 偶尔 |
| 📈 太多人用，限流了 | 经常 |
| 💸 你的额度用完了 | 看使用量 |
| 🌐 网络抖动 | 偶尔 |

**如果你只有一个模型，它挂了 = 你的 AI 团队全部瘫痪。**

---

## OpenClaw 的解决方案：Fallback 链

```json
{
  "model": {
    "primary": "deepseek-v3.2",         // 主力：深度求索
    "fallbacks": [
      "minimax-m2.7",                    // 替补1：MiniMax
      "siliconflow/deepseek-v3"          // 替补2：另一个通道的深度求索
    ]
  }
}
```

**工作原理**：

```
你发了一条消息给 AI
       │
       ▼
  DeepSeek V3.2 → 正常？→ 回答你 ✅
       │
      挂了？
       │
       ▼
  MiniMax M2.7 → 正常？→ 回答你 ✅
       │
      也挂了？
       │
       ▼
  SiliconFlow → 正常？→ 回答你 ✅
       │
    全挂了？
       │
       ▼
  "抱歉，所有模型都不可用" 😢（但这种情况极少极少）
```

### 对你来说体验是什么样？

**完全无感。** 你发消息，AI 回答。你根本不知道它偷偷换了模型。

就像你打车，滴滴自动帮你换了一辆车，你可能根本没注意。

## 更高级的玩法：多账号轮换

如果你特别担心额度用完：

```
账号 A 的额度用完了
    ↓ 自动切换
账号 B 继续用
    ↓ 也用完了？
账号 C 继续用
```

就像你有三张银行卡，一张刷爆了自动换下一张。

## 代理层架构：自建中转站

我在本地 Mac 上跑了 3 个"代理"：

```
                    ┌─────────────────────┐
                    │  你的 Mac            │
                    │                     │
AI 请求 ──────────→ │  端口 18790          │ ──→ MiniMax 服务器
                    │  （MiniMax 代理）     │
                    │                     │
AI 请求 ──────────→ │  端口 18793          │ ──→ SiliconFlow 服务器
                    │  （SiliconFlow 代理）│
                    │                     │
AI 请求 ──────────→ │  端口 18792          │ ──→ 52codex 服务器
                    │  （Relay 代理）      │
                    └─────────────────────┘
```

**为什么要自建代理？**

1. 把不兼容的 API 格式"翻译"成统一格式
2. 可以加日志，追踪每次调用
3. 挂了可以本地重启，不依赖外部

---

## 📝 总结

| 策略 | 效果 |
|------|------|
| Fallback 链 | 一个模型挂了自动换下一个 |
| 多账号轮换 | 额度用完自动换账号 |
| 本地代理 | 统一 API 格式，加日志 |
| **最终效果** | **AI 团队 24 小时在线，几乎不会全挂** |
